TEXI Converter (to/from PNG) for 3DS game 「NEWラブプラス+」/「NEWLOVEPLUS+」.

Drag & Drop to convert a TEXI to PNG and back to TEXI.

References: parts of the code for this tool origins from Ohana3DS Rebirth & png2bclim:
https://github.com/gdkchan/Ohana3DS-Rebirth
https://github.com/kwsch/png2bclim/blob/master/png2bclim/BCLIM.cs
